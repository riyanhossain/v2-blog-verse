Simple blogging web app

## Run the Application



### Server-Side
make sure you are in the ```root``` directory.
```bash
npm install or npm i
```

### Client-Side
make sure you are in the ```root/client``` directory.
```bash
npm install or npm i
```

## Usage

```
npm run dev
```
